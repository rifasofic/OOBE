# OOBE

Out of the Box Clea Experience

## Requirements

- nodejs  >= 24.6.0

## Installation and Setup Instructions

### Navigate to project:

```sh
cd oobe
```

### Install NPM packages:

```sh
npm install
```

### Start the app:

```sh
npm run start
```

### Visit app:

[http://localhost:5173/](http://localhost:5173/)
